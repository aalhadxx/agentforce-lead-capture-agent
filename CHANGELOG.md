# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-12
### Added
- Initial release of the Agentforce Lead Capture Agent.
- `LeadCaptureAgent` Agent Script DSL definition.
- `LeadCaptureTurnAction` Apex invocable action to parse messages, maintain state, and create Leads.
- `LeadCreationAction` fallback Apex invocable action.
- `LeadCaptureAgent_Apex_Access` permission set for the Einstein Service Agent user.
- Comprehensive `README.md` and `CLAUDE.md` with explicit `sf` CLI deployment and testing instructions.
- Community health files (`CONTRIBUTING.md`, Issue Templates, PR Template).
